#Хоткеи
<kbd>Ctrl</kbd> + <kbd>E</kbd> - выйти из редактора

<kbd>Ctrl</kbd> + <kbd>Q</kbd> - закрыть текущий файл

<kbd>Ctrl</kbd> + <kbd>S</kbd> - сохранить изменения

<kbd>Ctrl</kbd> + <kbd>Tab</kbd> - следующий файл
